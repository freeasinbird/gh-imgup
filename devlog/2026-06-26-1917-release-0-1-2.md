# Release 0.1.2

Branch `chore/release-0.1.2`. Cut the third release, bundling the changes
merged since `v0.1.1` (#40 lockfile sync, #41 skill-trigger delineation,
#42 `--help` review-checklist expansion).

## Decisions

- **Patch bump (0.1.1 → 0.1.2).** All changes since v0.1.1 are docs / skill /
  help-text — no CLI surface or machine-output-contract change (invariant 7),
  so a patch is correct under the 0.x policy. Not a minor; nothing new to call.
- **`npm version 0.1.2 --no-git-tag-version`** bumps `package.json` and
  `package-lock.json` together — avoids the 0.1.1 lockfile-drift bite the
  AGENTS.md gotcha warns about. Verified both root and `packages[""]` show
  `0.1.2` / `@freeasinbird/gh-imgup`.
- **CHANGELOG:** promoted `[Unreleased]` → `[0.1.2] - 2026-06-26`, left an empty
  `[Unreleased]`, re-pointed the compare links. Added a skill-trigger bullet
  (#41) the `--help` PR hadn't logged; the lockfile sync (#40) is internal and
  stays out of the user-facing changelog.
